Module Flashcards.Card

module Card: sig .. end
This module covers a single flashcard. This small library contains the usual accessing and setting functions you use to manipulate a flashcard.

The code is trivial and shouldn't be difficult to understand.

type t = {
   mutable front : string;
   mutable back : string;
}
A Card.t contains a front and a back.
val front : t -> string
val back : t -> string
val to_string : t -> string
val set_front : t -> string -> unit
val set_back : t -> string -> unit
val create : string -> string -> t
val reverse : t -> unit
Reverse or swap the front and back values.

This is often used with flashcards for foreign languages where one wants to practice with say, English to Latin and then Latin to English. This also let's you play "Jeopardy."
